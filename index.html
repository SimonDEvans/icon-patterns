Icon patterns for your background

Append animated icons in the background of any DOM element. Just import use any iconic font set that you prefer and the library will inject icons with random animations in your DOM element. You are currently viewing an example in the top section.

Import Dependencies

Import the font icons you want in your head. In this example we are using Ionicons.com. 

<link href="https://cdnjs.cloudflare.com/ajax/libs/ionicons/2.0.1/css/ionicons.cssfrel="stylesheet" type="text/css" >

Load the css assets in the head section. 

<link href="./dist/icon-patterns.css" rel="stylesheet" type="text/css">

Load the javascript assets before closing the body tag. 

<!-- jQuery import -->
<script src="https://code.jquery.com/jquery-3.2.1.min.js"></script>
<!-- IconPatterns import -->
<script src="./dist/icon-patterns.min.js" ></script>

Setup DOM 

<div id="my-container"> We've created Icon Patterns </div>

Configure Icon patterns

Include the name prefixes as described in the font icon set that you have imported. 

$('#my-container').IconPatterns({
  color: '#FFFFFF',
  icons: [{
    'ion-cube': {
      count: 10,
      sizeVariation: [10, 30],
    },
    'ion-erlenmeyer-flask': {
      count: 15,
      sizeVariation: [5, 20],
    }]
  }]
}).draw();
